# Cabinet Design Studio

پروژه‌ی نمونه طراحی کابینت آشپزخانه با:

- Next.js 15 (App Router)
- shadcn/ui components
- Three.js با react-three-fiber + drei

## اجرا

```bash
npm install
npm run dev
```

سپس:

```text
http://localhost:3000
```

## قابلیت‌ها

- محیط کاملا سه‌بعدی با دوربین Orbit
- ماژول‌های کابینت پارامتریک (زمینی / هوایی / ایستاده)
- تنظیم تعداد یونیت، بازشدگی درب، متریال
- زیرساخت مناسب برای توسعه BOM، اکسسوری، متره و استانداردهای ارگونومی
